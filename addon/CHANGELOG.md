# Changelog

## 0.1.0

- Initial Home Assistant App repository for hass-vue.
- Builds one independent Vue/Vite module per page folder.
- Writes stable Home Assistant `/local/hass-vue` output with marker-gated deletion.
- Adds status HTML/JSON output, watcher daemon, example page, and validation script.
- Reframed output as generic Vue browser modules with Home Assistant custom element adapters.
